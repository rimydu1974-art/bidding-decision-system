| 功能 | 说明 |
|---|---|
| **AI速读** | 上传招标文件（PDF/Word/扫描件），AI深度解析资质要求、评分标准，生成老板决策摘要 |
| **智能风控** | 废标风险自动发现，合规性一键检测，缺失材料智能提醒 |
| **6大核心指标** | 全景扫描，每条结论附带引用来源，支持响应性审查、资质合规校验、报价合理性比对 |

### 产品亮点

- **3分钟**：200页标书 → 1页决策摘要
- **99%**：废标隐患拦截率
- **200+**：风险规则库

## 第四章 上传与分析

### 完整流程

```
上传招标文件 → AI自动分析 → 查看分析结果 → 上传投标文件（可选）→ 深度评估
```

### 第一步：上传招标文件

1. 点击 **「新建项目」** 或 **「上传招标文件」**
2. 选择招标文件（支持 PDF、Word 格式）
3. 等待文件上传完成

### 第二步：AI自动分析

1. 文件上传后，系统自动启动AI分析
2. 页面显示分析进度条
3. 分析时间取决于文件大小，通常 1-3 分钟

### 第三步：查看分析结果

分析完成后，自动进入项目详情页，展示：
- **投标决策表**（6大类信息提取）
- **老板总结**（是否参与、核心门槛、制胜策略）
- **偏离表**（招标要求 vs 投标响应）
- **风险评估**（废标风险、高/中/低分级）
- **评分预测**（评分项拆解、得分预估）

### 第四步：上传投标文件（可选）

1. 在项目详情页，点击 **「上传投标文件」**
2. 选择您的投标文件
3. 系统自动对比招标文件和投标文件
4. 生成深度评估报告（偏离分析、差距提醒）

## 第五章 查看分析结果

### 投标决策表（6大类）

| 类别 | 内容 |
|---|---|
| **第1类：基本信息** | 项目名称、编号、招标企业、联系人、代理机构等 |
| **第2类：财务信息** | 预算金额、最高限价、付款方式、保证金等 |
| **第3类：资质要求** | 联合体投标、特殊资质、资格性审查等 |
| **第4类：评分规则** | 总分、客观分/主观分、价格分、废标说明等 |
| **第5类：时间要求** | 开标时间、提问截止、交货时间等 |
| **第6类：项目信息** | 技术需求、废标条件、密封要求等 |

### 老板总结

为管理层提供快速决策依据：
- **是否建议投标**
- **核心门槛**（必须满足的条件）
- **制胜策略**（差异化竞争点）
- **风险提示**（需关注的关键问题）

### 偏离表

对比招标要求与投标响应，标注：
- ✅ 完全满足
- ⚠️ 部分偏离
- ❌ 不满足

### 风险评估

按严重程度分级：
- 🔴 **严重风险**：可能导致废标
- 🟠 **高风险**：需重点关注
- 🟡 **中等风险**：建议优化
- 🟢 **低风险**：可接受

1. 在项目详情页，点击右上角 **「导出报告」** 按钮

## 第十章 常见问题

### Q：免费额度用完了怎么办？

A：您可以：
- 等待下月额度重置（每月1日自动重置）
- 配置自己的AI API Key（DeepSeek/通义千问/智谱等）
- 升级为专业版享受不限额度

### Q：如何配置自己的API Key？

A：进入 **「用户中心」** → **「API Key配置」**，填入您的API Key即可。专业版/企业版用户必须配置自己的API Key。

### Q：导出的报告格式是什么？

A：
- Excel格式：投标决策评估表（.xlsx），可用Excel/WPS编辑
- PDF格式：排版报告，色块分区，适合打印分享

### Q：免费用户可以导出吗？

A：可以。免费用户可导出投标决策评估表，但老板总结部分仅显示数量预览。

### Q：单次版¥19有时间限制吗？

A：购买后7天内可导出完整报告，7天后无法导出，但仍可查看分析结果。

### Q：如何上传投标文件进行深度评估？

A：在项目详情页，分析结果下方有「上传投标文件」入口，上传后系统自动对比分析。

### Q：案例中心的案例安全吗？

A：所有用户上传的案例都会经过自动脱敏处理，隐藏敏感信息后才展示。
